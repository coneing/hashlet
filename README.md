## Installation
1. **Clone the Repository**
   ```sh
   git clone https://github.com/coneing/hashlet.git
   cd hashlet
   ```

2. **Set up Virtual Environment**
   ```sh
   python -m venv hashlet_env
   source hashlet_env/bin/activate # For macOS/Linux
   hashlet_env\Scripts\activate # For Windows
   ```

3. **Install Dependencies**
   ```sh
   pip install -r requirements.txt
   pip install greenlet numpy matplotlib cryptography flask flask-socketio python-dotenv networkx
   ```

4. **Compile Reaper (C)**
   ```sh
   gcc reaper.c -o reaper -lcrypto
   ```

## Usage
### Running Blocs每m CLI (Main Entry)
```sh
python blocsym.py
```
Idle loop with key moshing; enter keys to mosh, Ctrl+C to exit. Demos: execute_function_string('BFS', file_path='example.txt') or 'BASTION' for multi-bastion.

### Running as Server
```sh
python blocsym.py server
```
Flask + SocketIO on port 5000. Endpoints: /metrics/block_height, etc.

### Bloom Filter (bloom.py)
```sh
python bloom.py
```
Adds genesis prompt. Usage: seraph.add("your_prompt")

### Reaper (reaper.c)
